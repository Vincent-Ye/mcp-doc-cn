<Update label="2025-02-14" description="Java SDK 发布">
  - 我们很高兴地宣布，由 VMware Tanzu 的 Spring AI 团队开发的 Java SDK，现在成为 MCP 的官方 [Java SDK](https://github.com/modelcontextprotocol/java-sdk)。
    它加入了我们现有的 Kotlin SDK，成为我们支持语言列表中的一员。
    Spring AI 团队将作为 Model Context Protocol 组织的重要组成部分，维护此 SDK。我们很激动欢迎他们加入 MCP 社区！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDK 的版本 [1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1) 已发布，
    提供了重要的稳定性改进和漏洞修复。
</Update>

<Update label="2025-01-18" description="SDK 和服务器改进">
  - 在 [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) 中引入了简化的、类似 Express 的 API。
  - 在 [客户端页面](https://modelcontextprotocol.io/clients) 添加了 8 个新客户端。
</Update>

<Update label="2025-01-03" description="SDK 和服务器改进">
  - 在 [Python SDK](https://github.com/modelcontextprotocol/python-sdk) 中新增了 FastMCP API。
  - 在 [服务器仓库](https://github.com/modelcontextprotocol/servers) 中添加了基于 Docker 的 MCP 服务器。
</Update>

<Update label="2024-12-21" description="Kotlin SDK 发布">
  - Jetbrains 发布了针对 MCP 的 Kotlin SDK！
  - 想了解 MCP Kotlin 服务器示例，可查看[此仓库](https://github.com/modelcontextprotocol/kotlin-sdk/tree/main/samples/kotlin-mcp-server)。
</Update>
